# 歷屆物理詳解校訂與發佈標準

本專案的 `index.html` 只有在所有年度、所有題目都通過下列檢查後，才可視為最終發佈版。

## 逐題詳解標準

1. 若題目含數值計算，詳解必須寫出完整計算過程：
   - 先列出使用的物理模型或定律。
   - 寫出公式。
   - 代入題目數值。
   - 保留必要單位換算。
   - 說明每個選項為何正確或錯誤，尤其是多選題。

2. 若題目是原理或觀念判斷，詳解必須寫出推演過程：
   - 說明判斷依據。
   - 說明適用條件。
   - 逐項排除錯誤選項。
   - 不能只寫「依定律判斷」或只給答案。

3. 詳解中的公式需使用出版式科學符號：
   - 使用 `π`，不用 `Pi`。
   - 使用 `4π²`，不用 `4Pi^2` 或 `4π^2`。
   - 使用 `r²`、`T²`、`v₀`、`P₀` 等上標與下標。
   - 使用 `Δt`、`λ`、`θ`、`ω`、`μ` 等科學符號。

4. 題目截圖需完整呈現：
   - 每題截圖要包含完整題幹、圖表、公式與所有選項。
   - 選擇題若有五個選項，截圖需包含 A-E。
   - 截圖不可切到下一題第一行。

5. 發佈前必須執行稽核：
   - `tools/audit_solution_quality.py`
   - `tools/check_publish_ready.py`
   - `tools/build_question_shots.py` 的截圖檢查結果需無缺漏。

## 發佈門檻

`data/solution-quality-audit.json` 的下列數字必須全部為 0：

- `flagged`
- `genericOrPlaceholder`
- `typewrittenFormulaStyle`

在未達成前，頁面只能視為「校訂中草稿」，不得標示為最終發佈版。
